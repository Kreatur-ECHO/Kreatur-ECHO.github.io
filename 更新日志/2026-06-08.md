## 💬 评论系统四次迭代

| 版本 | 方案 | 问题 |
|------|------|------|
| v1 | **Giscus**（第三方评论） | 懒加载导致首屏不显示 |
| v2 | Giscus + DOM API 创建 script | `innerHTML` 不执行 script |
| v3 | Giscus + 本地回退（Firefox/Edge 降级） | 第三方被浏览器拦截 |
| v4 | **GitHub Issues API 直读** ✅ | 初始方案没解决速率限制 |

### 最终方案（v4）
- 浏览器直接从 GitHub Issues API (#2) 拉取评论（实时，零延迟）
- GitHub Action `fetch-comments.yml` 每 10 分钟备份到 `data/comments.json`
- API 不可用时回退到本地静态 JSON
- 完全自建，不依赖任何第三方服务

## 🔧 早期修复

- Giscus 懒加载首屏不显示 → 移除 `loading="lazy"`
- `innerHTML` 不执行 `<script>` → 改用 DOM API 创建 script 元素
- Firefox 兼容性 → 添加降级提示（后续完全移除 giscus）
- 评论 API 速率限制（60/hr）→ GitHub Action 定时拉取静态 JSON
