# 更新日志 — 2026-06-10

## 修复

### GitHub API 速率限制 → 评论不更新
- **原因**：页面每次加载发 3 次未认证 GitHub API 请求（用户信息 + 仓库列表 + 评论），共享 60 次/小时/IP 配额
- **修复**：`js/github.js`（v7→v8）和 `js/comments.js`（v12→v13）添加 `authHeaders()` 函数，使用 `SiteConfig.reactionsToken` 发认证请求，配额 60→5000/hr
- **文件**：`js/github.js`、`js/comments.js`、`index.html`

### COS 写入 403
- **原因**：`signKey` 用 `.digest()` 得到 Buffer，签名不一致
- **修复**：改用 `.digest('hex')` 得到 hex 字符串
- **文件**：`scf/app.js`

### COS 签名不匹配
- **原因**：签了 `content-type` 头（不在 SDK 白名单）
- **修复**：只签 `host` 头
- **文件**：`scf/app.js`

### 点赞 IP 去重失效
- **原因**：`x-forwarded-for` 取了整个代理链 `ip1, ip2, ip3`，每次请求 MD5 不同
- **修复**：`split(',')[0].trim()` 只取第一个真实客户端 IP
- **文件**：`scf/app.js`

### 已点赞按钮 Firefox 不响应
- **原因**：`disabled` 属性阻止 Firefox 点击事件冒泡
- **修复**：改用 class `already-liked`
- **文件**：`js/comments.js`、`css/style.css`

### 展开卡片排版错乱
- **原因**：`display:contents` 兼容问题 + `display:block` 覆盖了 `display:flex`
- **修复**：直接给 `<a>` 加 class，改为 `display:flex`
- **文件**：`js/renderer.js`、`css/style.css`

### test/test2 文章丢失
- **原因**：git merge 冲突解决失误
- **修复**：手动恢复文章数据
- **文件**：`data/posts.js`
