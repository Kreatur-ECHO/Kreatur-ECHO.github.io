# 更新日志 — 2026-06-12

> 设计主页入场动画原型，经历 CSS animation 与 hover transition 冲突的长调试，最终确立架构分离方案。

## 入场动画原型开发

### 设计目标
- 进入主页时蒙上更黑的遮罩层
- 米色圆月从底部居中升起，停至头像高度
- 月亮 hover 轻微放大（与头像交互一致）
- 3D 立体纹理波动、水波光晕、金粉碎光
- 仅点击月亮解除遮罩

### 技术难点：CSS animation 与 hover transition 冲突
- 月亮入场动画用 `transform: scale()` → 动画结束后 `transform` 被 animation 锁定
- hover 的 `transition: transform` 无法覆盖 animation
- **解决方案**：上升动画放在外层容器 `.moon-stage`（只动 `bottom`），月亮本体 `.intro-moon` 不参与动画，transition 干干净净

### 最终架构

| 元素 | 职责 | 动画方式 |
|------|------|----------|
| `.intro-overlay` | 全屏黑色遮罩 | `opacity 0.6s` 解散 |
| `.moon-stage` | 月亮容器 | CSS `moonRise` — 只动 `bottom` 位置 |
| `.moon-stage::before` | 底层柔光 | 静态径向渐变 |
| `.moon-stage::after` | 水波环 | `rippleIn` 向心收缩 3s 循环 |
| `.intro-moon` | 圆月本体 | **纯 transition** — hover `scale(1.08)` |
| `.moon-texture` ×2 | 3D 月面纹理 | `textureDrift` 双轨反向漂移 + rotate |
| `.moon-bulb` | 月芯霓虹 | 4 层 box-shadow 脉动 2.5s |
| `.intro-spark` ×18 | 金粉碎光 | `sparkRise` 从月下方向上飘散 |

### 视觉细节
- 月面：`radial-gradient` 从米白到暗金 4 层渐变色
- 纹理：多个椭圆形暗纹/亮斑径向渐变，双轨反向漂移（5s / 6.5s）
- 芯灯：8px 纯白核 + 四层 box-shadow 扩散光晕，透明度脉动
- 金粉：18 颗，偏下方半圆散布，5-15s 随机时长循环上浮
